# Fillit
Form the smallest possible square from a list of tetriminos.

This code complies with the 42 school norm

## Compiling
Run `make` command and an executable called `fillit` should be created.

## Usage
```
./fillit [file]
```

## Example

```
./fillit test_file

ABBBB.
ACCCEE
AFFCEE
A.FFGG
HHHDDG
.HDD.G
```
